# Beginner Challenges

These challenges are designed in a meaningful way to learn and recall important concepts of Go.

## Scope

Through these challenges, you will practice:

- Structs and methods (value receivers, pointer receivers)
- Slices, `range`, and iteration
- Error handling (`errors.New`, `if err != nil`)
- File I/O (`os.Open`, `defer`, `Read` with buffers)
- Formatted output (`fmt.Printf`, `fmt.Println`)

In order to start building something appropriate in Go.

## Challenges

| # | Name | Type | Key Concepts |
|---|------|------|-------------|
| 1 | The Broken Student Report | Fix It 🔧 | Structs, methods, `range`, package declaration |
| 2 | The Beer Eligibility Checker | Build It 🏗️ | Structs, methods, slices, `fmt.Printf` |
| 3 | The File Word Counter | Build It 🏗️ | File I/O, `os.Open`, `defer`, byte buffers, `io.EOF` |
| 4 | The ATM Machine | Build It 🏗️ | Pointer receivers, error handling, slices of structs |

## Requirements

1. Try to solve it by yourself (99%), use as little AI as possible
2. Turn off all supporting code-generation tools

Example solutions are in the `solutions/` folder.
