# Fundamental Go

Learn Go project structure, basic syntax, and user-defined types.

## Source

https://www.youtube.com/watch?v=3lazW_dSXKM

## Design

After learning and experimenting, it's important to test my knowledge. For this purpose, I created 2 folders in this order:

1. **[go-flashcards](go-flashcards/)** — A React flashcard app with 26 questions across 4 difficulty levels (Bloom's Taxonomy) to recall and deepen understanding
2. **[beginner_challenges](beginner_challenges/)** — 4 progressive Go coding challenges (fix-it, build-it, apply-it) with solutions

Enjoy!
